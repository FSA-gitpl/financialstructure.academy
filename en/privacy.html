Privacy Policy

This Privacy Policy describes the rules for data processing on the Financial Structure Academy (“FSA”) website. The website is informational in nature and does not collect personal data except in the cases described below.

1. Scope of Data Processing

FSA does not maintain user accounts and does not actively collect personal data. Data may be processed only when a user contacts FSA via e‑mail.

2. E‑mail Contact

By sending a message to office@financialstructure.academy, the user voluntarily provides data. This data is used solely to respond to the inquiry.

3. Cookies

The website may use basic cookies necessary for proper functionality. FSA does not conduct user profiling.

4. Data Sharing

Data is not shared with third parties unless required by applicable law.

5. User Rights

Users may request information about data processing, correction of data, or deletion by contacting FSA via e‑mail.

6. Contact

For matters related to the Privacy Policy, please contact: office@financialstructure.academy
